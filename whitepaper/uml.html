4. Agent Decision Control Flowchart

This flowchart maps the internal branch decisions executed inside the autonomous agent pipeline (agent.php) until completion.

flowchart TD A([Start Audit]) --> B[Read file contents & extract module parameters] B --> C[Fetch MLO Baselines & Assessments from MySQL] C --> D[Initialize Conversation History & System Guidelines] D --> E[Call Fireworks AI GLM-5p2] E --> F{Output contains XML call?} F -- Yes: web_audit --> G[Parse query & call Brave Search API via cURL] G --> H[Inject Brave results into context] H --> E F -- Yes: save_audit_result --> I[Parse JSON payload, score, recommendations] I --> J[Write record to MySQL audit_results] J --> K([End Audit - Notify Dashboard]) F -- No XML / Text answer --> L{Loop count < 5?} L -- Yes --> M[Inject directive to use XML tool calls] M --> E L -- No --> N[Force fallback JSON parsing] N --> J
